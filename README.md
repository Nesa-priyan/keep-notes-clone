# 📝 Keep Notes Clone

A simple and beautiful notes application inspired by Google Keep.
This project allows users to create, pin, and manage notes with a clean and minimal user interface.

The application is built using basic web technologies and is deployed publicly so anyone can try it directly from the browser.

---

## 🌐 Live Website

You can try the application here:

**Live Demo:**
https://nesa-priyan.github.io/keep-notes-clone/

Open the link in your browser and start writing notes instantly.

---

## ✨ Features

* Create notes instantly
* Pin important notes
* Clean and colorful UI
* Notes saved in browser storage
* Works on desktop and mobile browsers
* No login required

Each visitor can create their own notes, which are stored locally in their browser.

## ⚙️ How It Works

1. The user writes a note.
2. JavaScript captures the note.
3. The note is saved in the browser using Local Storage.
4. When the page reloads, the saved notes are loaded again.

Each user has their own notes stored on their device.
